# ⏳ Time is Money — 时薪实时计算器

> 你的每一秒，都值钱。

一个纯前端的时薪实时计算器，把枯燥的月薪换算成每秒跳动的数字。看着钱一秒一秒地涨，上起班来更有动力 😎

微信小程序版本已上线，请在微信进行搜索：每秒都值钱。即可进行使用！

## ✨ 功能

- **🕐 实时时钟** — 当前时间每秒刷新，精确到秒
- **💰 今天已赚** — 根据时薪实时计算你今天赚了多少，数字每秒跳动
- **📊 进度条** — 可视化今日工作完成进度
- **🏢 多种工作制度** — 双休 / 大小周 / 单休 / 月末单休 / 自定义天数
- **🎁 年终奖计入** — 12~20+ 薪任选，时薪 = (月薪 × 年薪月数 ÷ 12) ÷ 月工作小时
- **🍱 午休扣除** — 设置午休时段，午休期间收入暂停，进度条自动扣除
- **⏳ 倒计时** — 显示距离上班/午休/下班的倒计时
- **📅 本月累计** — 每天下班自动记录，统计本月已赚总额和工作天数
- **⏯️ 暂停/继续** — 随时冻结计数，适合摸鱼时暂停（不是）
- **💾 自动保存** — 配置和月度数据都存 localStorage，刷新不丢失
- **🌓 暗色/亮色** — 右上角一键切换深色/浅色主题，跟随系统偏好

## 🚀 使用

直接浏览器打开 `salary-calculator.html` 即可，无需任何服务器或依赖。

或者部署到任意静态托管：

```bash
# 比如用 Python 快速启动本地服务器
python -m http.server 8080
# 然后打开 http://localhost:8080/salary-calculator.html
```

## 🧮 计算逻辑

```
有效月薪 = 月薪 × 年薪月数 ÷ 12
时薪     = 有效月薪 ÷ (月工作天数 × 日工作小时)
每秒收入 = 时薪 ÷ 3600
实际工作秒 = 当前时间 − 上班时间 − 午休时长
```
